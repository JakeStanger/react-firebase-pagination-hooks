# react-firebase-pagination-hooks

This is heavily based on [firestore-pagination-hook](https://github.com/bmcmahen/firestore-pagination-hook) and [react-firebase-hooks](https://github.com/CSFrequency/react-firebase-hooks).
modified firestore-pagination-hook so that we cab use it like react-firebase-hooks

## Install

```
yarn add react-firebase-pagination-hooks
```

```
npm i react-firebase-pagination-hooks
```
